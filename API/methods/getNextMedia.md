# Get Next Media

Returns the user's next song in their playlist.

### Code

```js
API.getNextMedia();
```

### Response

```js
{
    media: {
        author: "Sleep Party People",
        cid: "aFUzvbkEvRk",
        duration: 441,
        format: 1,
        id: "0",
        image: "//i.ytimg.com/vi/aFUzvbkEvRk/default.jpg",
        title: "I'm Not Human At All",
    },
    inHistory: false
}
```
