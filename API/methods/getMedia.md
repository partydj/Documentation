# Get Media

Returns the object of the current playing song.

### Code

```js
API.getMedia();
```

### Response

```js
{
    author: "Sleep Party People",
    cid: "aFUzvbkEvRk",
    duration: 441,
    format: 1,
    id: "0",
    image: "//i.ytimg.com/vi/aFUzvbkEvRk/default.jpg",
    title: "I'm Not Human At All",
}
```
