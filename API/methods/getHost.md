# Get Host

Returns the host user object or `undefined`.

### Code

```js
API.getHosts();
```

### Response

```js
{
    id: 80,
    username: '',
    avatarID: '',
    language: 'en',
    joined: undefined,
    level: 1,
    role: 0,
    gRole: 0,
    badge: undefined,
}
```
